# Oxide

Highly adjustable, [Nitro](https://nitro.build/) based, modern Svelte Web Framework.

## Features

- **Nitro based** - Leverage Nitro's powerful server capabilities.
- **Reasonably opinionated** - Start with a solid foundation and customize as needed.
- **It's still Vite** - Leverage Vite's features and plugins to build your application.
- **Familiar** - Based on the paradigms and patterns you probably already know.
- **Hybrid Side** - It's up to you if code happens on the server or client.

## Create project

Using NPM:

```sh
npx giget@latest gh:oxidejs/oxide/templates/minimal my-app
```

Using Bun:

```sh
bunx giget@latest gh:oxidejs/oxide/templates/minimal my-app
```
